# BankApp – Console Banking System

## توضیحات پروژه
این پروژه یک اپلیکیشن بانکی ساده است که در محیط **کنسول** اجرا می‌شود.  
کاربران می‌توانند ثبت‌نام کنند، وارد حساب شوند و عملیات بانکی مثل واریز، برداشت و کارت‌به‌کارت انجام دهند.  
برای ذخیره‌سازی اطلاعات کاربران از **ArrayList<String[]>** استفاده شده است.

## امکانات
- **ثبت‌نام کاربر جدید** همراه با اعتبارسنجی  
  - یوزرنیم یکتا  
  - پسورد معتبر (حرف کوچک، بزرگ، عدد، کاراکتر خاص، حداقل ۸ کاراکتر)  
  - ایمیل معتبر (فرمت `*@aut.com`)  
  - شماره تلفن معتبر (۱۱ رقمی و شروع با 09)  
  - تولید شماره کارت ۱۶ رقمی یکتا با شروع 6037  
- **ورود و خروج کاربران**
- **نمایش موجودی حساب**
- **واریز و برداشت**
- **کارت‌به‌کارت بین کاربران ثبت‌شده**
- **خروج از برنامه**

## نحوه اجرا
- پروژه را در IDE اجرا کنید.
- دستورات را مطابق نمونه‌ها در کنسول وارد کنید.  
  مثال:
register Sam Sami123! SamGhorbani 09123456789 sam@aut.com
login Sam Sami123!
deposit 1000

markdown
Copy code

## نکات مهم
- موجودی حساب به صورت رشته ذخیره شده و با `Double.parseDouble` مدیریت می‌شود.
- برای جلوگیری از تکرار کارت، شماره کارت تصادفی تولید و در لیست کاربران بررسی می‌شود.
- فقط زمانی می‌توان عملیات بانکی انجام داد که کاربر وارد حساب شده باشد.
